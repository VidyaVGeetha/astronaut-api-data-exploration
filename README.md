# Astronaut API Data Exploration (IBM Mini Project)

This mini project demonstrates how to collect and analyze data from a public API using Python. 
The project retrieves real-time astronaut data from the Open Notify API, processes the JSON response, 
and extracts insights such as the total number of people currently in space and their names.

This work was completed as part of the IBM Data Analyst learning program.
